## 🛠️ Como rodar os arquivos

Para rodar os scripts PHP localmente:

1.  Tenha o PHP instalado (ou utilize o XAMPP/Laragon).
2.  Abra o terminal na pasta do arquivo desejado.
3.  Execute o comando:

```bash
php nome_do_arquivo.php
```

Para arquivos web, inicie o servidor embutido do PHP:

```bash
php -S localhost:8000
```
